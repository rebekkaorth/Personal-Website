MetaIntervals

MetaIntervals is the first App I have developed. I designed the user interface with Adobe Experience Design and used Xcode 8 as my IDE.
The app has a build in timer which allows the user to do e.g. interval training. In order to start the training the user needs to set the three sliders to the desired values.
MetaIntervals also includes an Apple Watch app.
